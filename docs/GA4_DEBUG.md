# Диагностика проблем с GA4 событиями

## 🔍 Если события не появляются через 2+ часа

### Шаг 1: Проверьте переменную окружения

#### В .env.local:
```env
NEXT_PUBLIC_GA_ID=G-7K87D6NR9E
```

#### В Vercel:
1. Откройте [Vercel Dashboard](https://vercel.com/dashboard)
2. Ваш проект → **Settings** → **Environment Variables**
3. Проверьте, что есть `NEXT_PUBLIC_GA_ID` со значением `G-7K87D6NR9E`
4. Если нет — добавьте и **Redeploy** проект

---

### Шаг 2: Проверьте в браузере (консоль)

1. **Откройте сайт** https://www.aidardev.kz
2. **Нажмите F12** (DevTools)
3. **Перейдите в Console**
4. **Введите команду:**
```javascript
window.gtag
```

**Ожидаемый результат:**
- Должна быть функция (не `undefined`)
- Если `undefined` — GA4 не загрузился

5. **Проверьте dataLayer:**
```javascript
window.dataLayer
```

**Ожидаемый результат:**
- Должен быть массив с данными
- Если `undefined` — проблема с инициализацией

---

### Шаг 3: Проверьте Network запросы

1. **F12** → **Network** tab
2. **Отфильтруйте:** `gtag` или `collect`
3. **Выполните действие на сайте** (откройте чат)
4. **Проверьте запросы:**
   - Должны быть запросы к `www.googletagmanager.com`
   - Должны быть запросы к `www.google-analytics.com/g/collect`

**Если запросов нет:**
- GA4 не загружается
- Проверьте блокировщики рекламы (AdBlock, uBlock)
- Проверьте переменные окружения

---

### Шаг 4: Проверьте отправку событий вручную

1. **Откройте консоль (F12)**
2. **Введите команду:**
```javascript
window.gtag('event', 'test_event', {
  event_category: 'Test',
  event_label: 'Manual test',
  value: 1
});
```

3. **Проверьте Network:**
   - Должен появиться запрос к `google-analytics.com/g/collect`
   - В URL должен быть параметр `en=test_event`

**Если запрос не появился:**
- `window.gtag` не работает
- Проблема с загрузкой GA4 скрипта

---

### Шаг 5: Проверьте Real-time отчет

1. **Откройте GA4** → **Reports** → **Real-time**
2. **Выполните действие на сайте:**
   - Откройте AI-чат
   - Кликните "Рассчитать стоимость"
   - Кликните "Связаться"
3. **Подождите 30-60 секунд**
4. **Проверьте раздел "Event count by Event name"**

**Если события есть в Real-time:**
- ✅ События отправляются правильно
- ⏰ Подождите 24-48 часов для появления в Events

**Если событий нет в Real-time:**
- ❌ События не отправляются
- Проверьте код отправки событий

---

### Шаг 6: Проверьте код отправки событий

Откройте консоль и проверьте:

```javascript
// Проверьте, что функция trackEvent работает
console.log(typeof window.gtag); // должно быть "function"

// Попробуйте отправить событие вручную
if (window.gtag) {
  window.gtag('event', 'chatbot_open', {
    event_category: 'Engagement',
    event_label: 'AI Chat',
    value: 1
  });
  console.log('Событие отправлено!');
} else {
  console.error('window.gtag не найден!');
}
```

## 🐛 Частые проблемы

### Проблема 1: События не отправляются

**Причины:**
- ❌ `NEXT_PUBLIC_GA_ID` не установлен
- ❌ Блокировщик рекламы (AdBlock)
- ❌ GA4 скрипт не загружается
- ❌ Ошибка в коде

**Решение:**
1. Проверьте переменные окружения
2. Отключите блокировщики
3. Проверьте консоль на ошибки
4. Перезапустите проект

---

### Проблема 2: События есть в Real-time, но нет в Events

**Причина:**
- ⏰ GA4 обрабатывает события с задержкой
- События появляются в Events через 24-48 часов

**Решение:**
- Подождите 24-48 часов
- Используйте Real-time для проверки

---

### Проблема 3: window.gtag is undefined

**Причины:**
- ❌ GA4 скрипт не загрузился
- ❌ Компонент GoogleAnalytics не рендерится
- ❌ Ошибка загрузки скрипта

**Решение:**
1. Проверьте, что компонент `GoogleAnalytics` добавлен в `layout.tsx`
2. Проверьте Network на ошибки загрузки скрипта
3. Проверьте переменную `NEXT_PUBLIC_GA_ID`

---

## ✅ Быстрая проверка

Выполните эти команды в консоли браузера:

```javascript
// 1. Проверка GA4
console.log('GA ID:', process.env.NEXT_PUBLIC_GA_ID); // undefined в браузере - это нормально
console.log('gtag:', typeof window.gtag); // должно быть "function"
console.log('dataLayer:', window.dataLayer); // должен быть массив

// 2. Отправка тестового события
if (window.gtag) {
  window.gtag('event', 'debug_test', {
    event_category: 'Debug',
    event_label: 'Test from console',
    value: 1
  });
  console.log('✅ Тестовое событие отправлено!');
} else {
  console.error('❌ window.gtag не найден!');
}

// 3. Проверка последних событий в dataLayer
console.log('Последние события:', window.dataLayer.filter(item => item[0] === 'event'));
```
