# Что делать после подключения Google Analytics 4

## ✅ Шаг 1: Проверьте, что Measurement ID добавлен

### В .env.local файле:
Откройте `.env.local` в корне проекта и убедитесь, что есть строка:
```env
NEXT_PUBLIC_GA_ID=G-7K87D6NR9E
```

### В Vercel:
1. Откройте [Vercel Dashboard](https://vercel.com/dashboard)
2. Ваш проект → **Settings** → **Environment Variables**
3. Проверьте, что есть переменная `NEXT_PUBLIC_GA_ID` со значением `G-7K87D6NR9E`
4. Если нет — добавьте и сохраните

## ✅ Шаг 3: Проверьте работу аналитики

### Способ 1: Real-time отчет (самый быстрый)
1. Откройте [Google Analytics](https://analytics.google.com)
2. Выберите ваш ресурс
3. Перейдите в **Reports** → **Real-time**
4. Откройте сайт https://www.aidardev.kz в новой вкладке
5. **Через 30-60 секунд** вы должны увидеть себя в Real-time отчете:
   - Количество пользователей: 1
   - Активные страницы: ваша текущая страница

### Способ 2: Проверка в браузере (DevTools)
1. Откройте сайт https://www.aidardev.kz
2. Нажмите **F12** (открыть DevTools)
3. Перейдите в **Network** tab
4. Отфильтруйте по `gtag` или `collect`
5. Вы должны увидеть запросы к:
   - `www.googletagmanager.com/gtag/js?id=G-7K87D6NR9E`
   - `www.google-analytics.com/g/collect?...`

### Способ 3: Расширение браузера
Установите [Google Analytics Debugger](https://chrome.google.com/webstore/detail/google-analytics-debugger/jnkmfdileelhofjcijamephohjechhna) для Chrome

- **Обычные отчеты:** через 24-48 часов после первого визита

## 🎯 Следующие шаги

### 1. Настройте конверсии в GA4 (ВАЖНО) ⏭️

**📖 Подробная инструкция:** [GA4_CONVERSIONS_SETUP.md](./GA4_CONVERSIONS_SETUP.md)

**Кратко:**
1. Откройте GA4 → **Admin** → **Events**
2. Найдите события: `chatbot_open`, `calculate_price_click`, `contact_button_click`
3. Включите переключатель **"Mark as conversion"** для важных событий
4. Проверьте через 24-48 часов в отчетах

**Приоритет конверсий:**
- 🔥 `chatbot_open` - ОБЯЗАТЕЛЬНО
- 🔥 `calculate_price_click` - ОБЯЗАТЕЛЬНО
- ⭐ `contact_button_click` - РЕКОМЕНДУЕТСЯ
- ⭐ `create_project_click` - РЕКОМЕНДУЕТСЯ

### 2. Подключите Google Search Console (РЕКОМЕНДУЕТСЯ) ⏭️

**📖 Подробная инструкция:** [GOOGLE_SEARCH_CONSOLE_SETUP.md](./GOOGLE_SEARCH_CONSOLE_SETUP.md)

**Кратко:**
1. Добавьте сайт в [Google Search Console](https://search.google.com/search-console)
2. Подтвердите владение (HTML-тег через `NEXT_PUBLIC_GOOGLE_VERIFICATION`)
3. Отправьте sitemap: `https://www.aidardev.kz/sitemap.xml`
4. Свяжите с GA4 для полной аналитики

### 3. Настройте аудитории (опционально) ⏭️

Создайте аудитории для таргетинга:
1. **Admin** → **Audiences** → **New audience**
2. Примеры:
   - **Потенциальные клиенты:** посетители с событием `chatbot_open`
   - **Заинтересованные:** посетители с событием `calculate_price_click`
   - **Высокая вовлеченность:** посетители с >3 страницами

- **Где:** Reports → Acquisition

### 3. Engagement (Вовлеченность)

- **Где:** Reports → Conversions
- **Зачем:** Сколько лидов получено, какие источники работают лучше

## ⚠️ Важные моменты

1. **Данные не появляются сразу:**
   - Real-time работает сразу
   - Обычные отчеты — через 24-48 часов

2. **Проверьте блокировщики:**
   - AdBlock может блокировать GA4
   - Проверьте в режиме инкогнито

3. **Cookie consent:**
   - Если используете GDPR, убедитесь, что GA4 загружается после согласия

4. **Проверка работы:**
   - Используйте Real-time отчет для быстрой проверки
   - Если не видите себя через 2 минуты — проверьте настройки
