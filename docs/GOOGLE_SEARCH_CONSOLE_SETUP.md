# Настройка Google Search Console

## 🎯 Зачем нужен Google Search Console

- Мониторинг индексации сайта
- Отслеживание позиций в поиске
- Анализ поисковых запросов
- Получение данных о кликах и показах
- Интеграция с Google Analytics 4
- Уведомления о проблемах

---

## 📝 Шаг 1: Добавьте сайт в Search Console

### 1. Откройте Google Search Console
- Перейдите на https://search.google.com/search-console
- Войдите в аккаунт Google

### 2. Добавьте свойство
- Нажмите **"Добавить ресурс"** или **"Add property"**
- Выберите тип: **URL-префикс** (рекомендуется)
- Введите URL: `https://www.aidardev.kz`
- Нажмите **"Продолжить"**

---

## 🔐 Шаг 2: Подтвердите владение сайтом

### Метод 1: HTML-тег (рекомендуется)

1. **Выберите метод:** HTML-тег
2. **Скопируйте код** (пример: `<meta name="google-site-verification" content="abc123..." />`)
3. **Добавьте в `.env.local`:**
   ```env
   NEXT_PUBLIC_GOOGLE_VERIFICATION=abc123...
   ```
4. **Проверьте в `app/layout.tsx`:**
   - Код уже должен быть в metadata.verification.google
   - Если нет - добавьте:
   ```typescript
   verification: {
     google: process.env.NEXT_PUBLIC_GOOGLE_VERIFICATION,
   },
   ```
5. **Добавьте в Vercel Environment Variables**
6. **Перезапустите проект**
7. **Вернитесь в Search Console** → Нажмите **"Проверить"**

### Метод 2: HTML-файл (альтернатива)

1. **Выберите метод:** HTML-файл
2. **Скачайте файл** (например, `google123abc.html`)
3. **Поместите в `public/`** папку
4. **В Search Console** → Нажмите **"Проверить"**

### Метод 3: DNS-запись (для продвинутых)

1. **Выберите метод:** DNS
2. **Добавьте TXT-запись** в настройках домена
3. **Подождите 24-48 часов**
4. **Проверьте**

---

## 📄 Шаг 3: Отправьте sitemap.xml

### 1. Найдите sitemap
Ваш sitemap доступен по адресу:
```
https://www.aidardev.kz/sitemap.xml
```

### 2. Отправьте в Search Console
1. В Search Console перейдите в **Sitemaps**
2. Введите: `sitemap.xml`
3. Нажмите **"Отправить"**

### 3. Проверьте статус
- Через несколько минут статус изменится на **"Успешно"**
- Если есть ошибки - исправьте их

---

## 🔗 Шаг 4: Свяжите с Google Analytics 4

### Преимущества связи:
- Данные о поисковых запросах в GA4
- Анализ эффективности SEO
- Понимание, какие запросы приводят к конверсиям

### Как связать:

1. **В Google Analytics 4:**
   - **Admin** → **Search Console linking**
   - Нажмите **"Link"**
   - Выберите свойство Search Console
   - Нажмите **"Confirm"**

2. **В Google Search Console:**
   - **Settings** → **Google Analytics property**
   - Выберите ваш GA4 ресурс
   - Сохраните

---

## 📊 Шаг 5: Настройте отчеты

### Отчет: Производительность
1. **Производительность** → **Результаты поиска**
2. Увидите:
   - Клики
   - Показы
   - Средняя позиция
   - CTR (кликабельность)

### Отчет: Индексация
1. **Индексация** → **Страницы**
2. Увидите:
   - Проиндексированные страницы
   - Исключенные страницы
   - Проблемы индексации

### Отчет: Поисковые запросы
1. **Производительность** → **Результаты поиска**
2. Фильтр: **Запросы**
3. Увидите, по каким запросам находят ваш сайт

---

## 🎯 Шаг 6: Настройте уведомления

1. **Settings** → **Users and permissions**
2. Добавьте email для уведомлений
3. Включите уведомления о:
   - Проблемах безопасности
   - Проблемах индексации
   - Новых ссылках

---

## 🔍 Шаг 7: Проверьте важные разделы

### 1. Покрытие (Coverage)
- **Индексация** → **Покрытие**
- Проверьте, нет ли ошибок индексации
- Исправьте проблемы, если есть

### 2. Улучшения (Enhancements)
- **Улучшения** → **Core Web Vitals**
- Проверьте производительность
- Оптимизируйте, если нужно

### 3. Безопасность
- **Безопасность и действия пользователей**
- Проверьте, нет ли проблем

---
