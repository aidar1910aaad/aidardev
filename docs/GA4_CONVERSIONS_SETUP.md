# Настройка конверсий в Google Analytics 4

## 🎯 Что такое конверсии

Конверсии — это важные действия пользователей на вашем сайте, которые приводят к бизнес-результатам (лиды, продажи, подписки и т.д.).

## ✅ События уже настроены в коде

В проекте уже добавлены события для отслеживания лидов:

### Основные события лидов:
- `contact_form_submit` - отправка формы контактов
- `chatbot_open` - открытие AI-чата
- `calculate_price_click` - клик "Рассчитать стоимость"
- `contact_button_click` - клик "Связаться"
- `create_project_click` - клик "Разработать проект"

### События вовлеченности:
- `project_view` - просмотр проекта
- `project_link_click` - клик по ссылке проекта
- `phone_call` - клик по телефону
- `whatsapp_click` - клик по WhatsApp
- `email_click` - клик по email
- `service_expand` - раскрытие услуги
- `faq_expand` - раскрытие FAQ
- `scroll_to_section` - прокрутка к секции

---

## 📝 Шаг 1: Проверьте, что события работают

### Способ 1: Real-time отчет
1. Откройте [Google Analytics](https://analytics.google.com)
2. Перейдите в **Reports** → **Real-time**
3. Выполните действие на сайте (например, откройте чат)
4. Через 30-60 секунд проверьте раздел **Event count by Event name**
5. Вы должны увидеть событие `chatbot_open`

### Способ 2: DebugView (рекомендуется)
1. В GA4 перейдите в **Admin** → **DebugView**
2. Откройте сайт в новой вкладке
3. Выполните действие (откройте чат, кликните кнопку)
4. В DebugView вы увидите события в реальном времени

---

## 🎯 Шаг 2: Настройте конверсии

### Вариант 1: Автоматически (через события)

1. **Откройте GA4:**
   - Перейдите в **Admin** → **Events**

2. **Найдите ваши события:**
   - `contact_form_submit`
   - `chatbot_open`
   - `calculate_price_click`
   - `contact_button_click`
   - `create_project_click`

3. **Включите как конверсии:**
   - Для каждого важного события нажмите переключатель **"Mark as conversion"**
   - ✅ `contact_form_submit` - **ОБЯЗАТЕЛЬНО**
   - ✅ `chatbot_open` - **ОБЯЗАТЕЛЬНО**
   - ✅ `calculate_price_click` - **РЕКОМЕНДУЕТСЯ**
   - ✅ `contact_button_click` - **РЕКОМЕНДУЕТСЯ**
   - ✅ `create_project_click` - **РЕКОМЕНДУЕТСЯ**

### Вариант 2: Создать кастомные конверсии

1. **Admin** → **Conversions** → **New conversion event**
2. Введите название события (например, `contact_form_submit`)
3. Нажмите **Create**

---

## 📊 Шаг 3: Настройте цели (Goals)

В GA4 цели настраиваются через конверсии:

1. **Admin** → **Conversions**
2. Убедитесь, что важные события отмечены как конверсии
3. GA4 автоматически будет отслеживать их как цели

### Рекомендуемые конверсии:

| Событие | Приоритет | Описание |
|:--------|:---------:|:---------|
| `contact_form_submit` | 🔥 Критично | Отправка формы - главная конверсия |
| `chatbot_open` | 🔥 Критично | Открытие чата - важный лид |
| `calculate_price_click` | ⭐ Важно | Интерес к расчету стоимости |
| `contact_button_click` | ⭐ Важно | Прямой контакт |
| `create_project_click` | ⭐ Важно | Создание проекта |
| `phone_call` | ⚠️ Средне | Звонок по телефону |
| `whatsapp_click` | ⚠️ Средне | Контакт через WhatsApp |

---

## 📈 Шаг 4: Настройте отчеты

### Отчет по конверсиям:
1. **Reports** → **Engagement** → **Conversions**
2. Вы увидите все конверсии и их количество

### Отчет по источникам конверсий:
1. **Reports** → **Acquisition** → **User acquisition**
2. Фильтр: **Conversions** → выберите вашу конверсию
3. Увидите, какие источники трафика дают больше конверсий

---

## 🎯 Шаг 5: Настройте аудитории

Создайте аудитории на основе конверсий:

### Аудитория: "Потенциальные клиенты"
1. **Admin** → **Audiences** → **New audience**
2. Название: "Потенциальные клиенты"
3. Условие: **Event** → `chatbot_open` OR `calculate_price_click`
4. Создайте

### Аудитория: "Высокая вовлеченность"
1. **New audience**
2. Название: "Высокая вовлеченность"
3. Условие: **Pages per session** > 3
4. Создайте

### Аудитория: "Просмотрели проекты"
1. **New audience**
2. Название: "Просмотрели проекты"
3. Условие: **Event** → `project_view`
4. Создайте

---

## 🔍 Проверка работы

### Через 24-48 часов после настройки:

1. **Reports** → **Engagement** → **Conversions**
   - Должны появиться данные о конверсиях

2. **Reports** → **Acquisition** → **User acquisition**
   - Увидите источники трафика с конверсиями

3. **Reports** → **Realtime** → **Conversions**
   - Real-time конверсии (если есть активность)

## 💡 Советы по оптимизации

### 1. Фокус на главных конверсиях
- Отслеживайте `contact_form_submit` и `chatbot_open` как основные
- Остальные — как вторичные метрики

### 2. Анализируйте воронку
- Сколько посетителей открыли чат?
- Сколько из них отправили форму?
- Где теряются пользователи?

### 3. Оптимизируйте на основе данных
- Если мало конверсий с `chatbot_open` → улучшите чат
- Если мало `contact_form_submit` → упростите форму
- Если много `calculate_price_click` → добавьте калькулятор

### 4. A/B тестирование
- Тестируйте разные тексты кнопок
- Тестируйте расположение CTA
- Тестируйте дизайн форм
