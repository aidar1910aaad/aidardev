## Рекомендуемый порядок (что лучше)

```
Блок 1: OLX починить          ← без этого реклама сольёт бюджет
    ↓
Блок 2: Фото + 2GIS + профили ← доверие ДО платной рекламы
    ↓
Блок 3: Google Ads + бонус   ← горячий спрос из поиска
    ↓
Блок 4: Таргет Instagram     ← малый бизнес KZ
    ↓
Блок 5: ИИ-услуги (упаковка) ← новый чек, интересные проекты
    ↓
Блок 6: Автоматизация         ← когда понятен поток лидов
    ↓
Блок 8: Стабильный поток       ← каждый месяц, не только реклама
    ↓
Блок 9: Скрипты продаж         ← не терять тех, кто уже написал
    ↓
Блок 10: Стратегия роста ЗП    ← директорский уровень, нестандартные ходы
    ↓
Блок 11: Outreach Factory       ← AI: КП + демо-сайт + бот → холодный аутрич (параллельно, по фазам)
```

**Почему фото раньше рекламы:** люди гуглят «разработчик астана» → видят объявление → заходят на сайт / WhatsApp. Лицо + отзывы на 2GIS = +30–50% к доверию. Без этого бонус Google уйдёт в клики без конверсии.

**Почему не биржи:** согласен — там гонка по цене с индусами за $200. Твои кейсы (QInvesting, Terricon) — уровень выше. OLX + Google + таргет + 2GIS лучше подходят.

## БЛОК 10 — Стратегия роста дохода (директорский уровень)

> **Главная идея:** ты зарабатываешь как сильный разработчик, а продаёшь себя как дешёвый OLX-исполнитель. Рост ЗП = **выше чек + меньше торга + повторяющиеся деньги + 1–2 жирных проекта в квартал**. Не «больше лидов».

### 10.1 Диагноз: где теряются деньги

| Что есть | Как используешь | Потеря |
|----------|-----------------|--------|
| 120+ проектов, QInvesting, Terricon | В OLX мелким шрифтом | Клиент видит «от 5000» |
| 6–7 слотов параллельно | Берёшь 3–5 по 80k | Простой или выгорание |
| 40 лидов/мес | 3–5 сделок | ~90% времени на мусор |
| Друзья-разрабы | «Иногда кидаю» | Нет студии = нет масштаба |

### 10.2 Три рычага роста (в порядке силы)

| Рычаг | Было | Стало | Эффект |
|-------|------|-------|--------|
| **1. Чек** | 80–100k | 130–180k | +60–80% при тех же 4 проектах |
| **2. Закрытие** | ~40% созвонов | ~60% | +1–2 проекта без новой рекламы |
| **3. Абонентка** | 0 | 3 × 25k | +75k ₸/мес «с потолка» |

**Математика по месяцам:**

| Этап | Расчёт | Выручка |
|------|--------|---------|
| Сейчас | 4 × 90k | ~360k (в удачные месяцы до 1M) |

| Через 4 мес | 3×150k + 1×350k + 75k | ~875k |
| + 1 «кит»/квартал 400k | в среднем +130k/мес | путь к **1.5M** |

### 10.3 Две линейки на OLX (Volume + Premium)

Не убирай города — **раздели объявления по уровню**, как Economy / Business:

| Линейка | Заголовок | Города | Для кого |

- [ ] Premium — другой первый ответ: «Расскажите о бизнесе», не «какой бюджет»

## БЛОК 11 — Outreach Factory (AI-фабрика холодного аутрича)

> **Идея:** система находит бизнесы без нормального сайта → AI делает **демо-сайт + демо-бота + КП** под них → ты отправляешь персональное сообщение: «Уже сделал набросок под ваш бизнес, посмотрите».

> **Статус:** [ ] концепт → MVP по фазам  
> **Когда начинать:** Фаза 0 — параллельно с Блоком 1. Полный пайплайн — после стабильных 600k+ или 2 недели на MVP.

### 11.1 Зачем это тебе (директорский взгляд)

| Обычный аутрич | Outreach Factory |
|----------------|------------------|
| «Здравствуйте, делаю сайты» | «Вот сайт **уже под ваш салон** — demo.aidardev.kz/salon-x» |
| 2% ответов | 15–30% ответов (конкретика + WOW) |
| Конкурируешь с OLX | Ты **единственный**, кто принёс готовое |
| Продаёшь 80k | «Доработать ваш демо под ключ — 120k» |

Это **нестандартное оружие**: ты dev + AI — 99% конкурентов на OLX так не могут.

**Плюс:** сам пайплайн потом продаёшь как услугу («AI-аутрич под ключ») — от 300k.

### 11.2 Что делает система (конвейер)

```
┌─────────────┐    ┌──────────────┐    ┌─────────────────────────────┐
│ 1. НАЙТИ    │ →  │ 2. АНАЛИЗ    │ →  │ 3. СГЕНЕРИТЬ               │
│ бизнесы     │    │ AI: ниша,    │    │ • лендинг (демо)            │
│ 2GIS/Google │    │ боль, УТП    │    │ • WhatsApp-бот (демо)       │
│ Instagram   │    │              │    │ • КП PDF                    │
└─────────────┘    └──────────────┘    └─────────────────────────────┘
                                                    ↓
┌─────────────┐    ┌──────────────┐    ┌─────────────────────────────┐
│ 6. СДЕЛКА   │ ←  │ 5. АУТРИЧ    │ ←  │ 4. ТЫ ПРОВЕРЯЕШЬ (15 мин)   │
│ доработка   │    │ WhatsApp /   │    │ approve / reject / edit     │
│ под ключ    │    │ Instagram    │    │ в админке                   │
└─────────────┘    └──────────────┘    └─────────────────────────────┘
```

**Важно:** сообщения отправляешь **ты**, не бот. Автомат — подготовка. Так не банят и не выглядит спамом.

### 11.3 Кого искать (критерии «идеального» бизнеса)

| Критерий | Почему |
|----------|--------|
| Нет сайта / только Instagram | Очевидная боль |
| Сайт старый / не открывается на телефоне | Легко показать «до/после» |
| Салон, клиника, кафе, автосервис, доставка | Понятный лендинг + бот заявок |
| Астана / Алматы (сначала) | Ты в Астане, знаешь рынок |
| Есть WhatsApp / Instagram для контакта | Куда писать |
| **Не** крупные сети / гос | Не ответят |

**Красные флаги (пропускать):**
- Уже нормальный сайт на Tilda/WordPress с заявками
